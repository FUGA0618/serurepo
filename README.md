# せるれぽとは
「いつか買おうと思っている書籍」がセールになったときに通知するサービスです。<br>
サイト上で書籍を検索してリストに追加しておくと、リスト内の書籍がセールになったときに通知します。<br>
もう、いろんなサイトでセールをやっていないか確認する必要はないし、書籍のセールを見逃すことはありません。
